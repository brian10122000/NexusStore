Variables à ajouter dans Railway/Cloudflare Dashboard :
  • STRIPE_SECRET_KEY → sk_live_xxxx (Stripe Dashboard → API keys)
  • SUMUP_API_KEY → clé API Sumup (me.sumup.com → Intégrations)
  • SITE_URL → https://votre-pseudo.github.io/nexusstore
  • SUPABASE_URL → votre URL Supabase
  • SUPABASE_SERVICE_KEY → clé service_role Supabase
