☁️ Supabase — Données cloud 24/7
⬤ Non connecté
✅
Avec Supabase, vos données (produits, avis, galerie) sont hébergées dans le cloud. Jamais perdues, accessibles depuis n'importe quel appareil.
URL Supabase
→ supabase.com → New Project → Settings → API → Project URL
Clé Anon (publique)
Settings → API → anon / public key
SQL à exécuter dans Supabase → SQL Editor
create table if not exists settings ( key text primary key, value text, updated_at timestamptz default now() ); alter table settings enable row level security; create policy "Public read" on settings for select using (true); create policy "Service write" on settings for all using (true);
⚡ Cloudflare Worker — Stripe & Sumup API
💡
Le Worker permet d'utiliser les vraies API Stripe et Sumup avec vos clés secrètes, stockées côté serveur (jamais visibles par les clients).
URL de votre Worker Cloudflare
C'est le même Worker que votre bot Discord. Le fichier worker-v5.js est fourni avec ce site.
Variables à ajouter dans Railway/Cloudflare Dashboard :
  • STRIPE_SECRET_KEY → sk_live_xxxx (Stripe Dashboard → API keys)
  • SUMUP_API_KEY → clé API Sumup (me.sumup.com → Intégrations)
  • SITE_URL → https://votre-pseudo.github.io/nexusstore
  • SUPABASE_URL → votre URL Supabase
  • SUPABASE_SERVICE_KEY → clé service_role Supabase
